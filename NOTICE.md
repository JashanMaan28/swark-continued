### Purpose of This Fork

This fork was created to ensure continued maintenance and development of the Swark extension. The goals of this fork include:

- Maintaining compatibility with the latest VS Code and GitHub Copilot updates
- Addressing bug fixes and community feature requests
- Continuing development of new features
- Preserving the AGPL-3.0 open-source license and spirit of the original project

### Attribution

All credit for the original concept, design, and implementation of Swark goes to the original Swark team at swark.io. This fork builds upon their excellent work.

Key contributions from the original project include:
- Core architecture diagram generation functionality
- Integration with GitHub Copilot and VS Code Language Model API
- Mermaid.js diagram generation and rendering
- File search and repository reading capabilities
- Automatic cycle detection and fixing in diagrams
- Support for multiple programming languages
- Telemetry infrastructure

### Fork-Specific Contributions

The following features and changes have been added in this fork:

- Dynamic model selection command with real-time GitHub Copilot model detection
- Comprehensive runtime validation for language model configuration
- Enhanced error messages with actionable feedback
- Three-tier fallback strategy for model selection
- Improved configuration flexibility by removing hardcoded model constraints
- Complete rebranding to "Swark Continued" to avoid confusion with the original project
- Updated documentation and contribution guidelines for the fork
